# Lineshapes in LARMOR — models, physics & literature

This is the reference for **every lineshape LARMOR can fit**: what it is, where it
comes from, the equation behind it, the primary literature, how you drive it in
the app, the conditions under which it is valid, and its limitations.

LARMOR's guiding principle is that a fitted lineshape should be a **physical
model**, not a curve that merely happens to overlay the data. Wherever the physics
is well defined, the model is computed from first principles — the quadrupolar and
shielding powder patterns and the disordered (Czjzek/CSA) distributions are
generated by the **mrsimulator** spin-dynamics engine (Srivastava & Grandinetti),
not by ad-hoc analytic shapes. Purely empirical shapes (Gaussian/Lorentzian, a
user function) are provided too, but are clearly labelled as such: they describe a
line, they do not explain it.

A recurring theme below is the distinction between **local** parameters that are
only meaningful for an ordered site (a single C_Q and η) and **rotational
invariants** — above all the *quadrupolar product* P_Q = C_Q·√(1+η²/3) — which are
the only quantities that keep their meaning once a site is disordered. Reporting
P_Q rather than a fictitious single C_Q for a glass is a deliberate choice, and it
follows directly from the Czjzek analysis (§ Czjzek).

## 1 · Analytic and empirical lines

### Gauss/Lorentz (pseudo-Voigt) — `gauss_lor`, `gl_norm`

**What it is.** A pseudo-Voigt: a linear blend of a Gaussian and a Lorentzian of
the *same* width and position, mixed by a single fraction `G/L` ($m$):

$$f(x) = (1-m)\,G(x) + m\,L(x)$$

$$G(x) = \exp\!\left[-4\ln 2\,\frac{(x-x_0)^2}{\Delta^2}\right] \qquad L(x) = \frac{1}{1 + 4(x-x_0)^2/\Delta^2}$$

both of FWHM $\Delta$ centred at $x_0$. $m = 0$ is pure Gaussian, $m = 1$ pure
Lorentzian. The two registered variants
differ only in normalisation: **`gauss_lor`** scales to peak height (dmfit's *GL*),
**`gl_norm`** scales to unit area (dmfit's *GL Norm*), so its amplitude is directly
proportional to the number of spins — the correct choice for quantification.

**History & literature.** The pseudo-Voigt is a computational stand-in for the
true Voigt profile (below), introduced because the Voigt has no closed form. The
form and its accuracy are analysed by Ida, Ando & Toraya, *J. Appl. Cryst.* **33**,
1311 (2000). It is the everyday line of dmfit (Massiot et al., *Magn. Reson. Chem.*
**40**, 70 (2002)).

**How it's used.** Set position, FWHM (in ppm or Hz — LARMOR accepts either), and
the G/L fraction. Gaussian character models **inhomogeneous** broadening (site
disorder, susceptibility, B₀ inhomogeneity); Lorentzian character models
**homogeneous** broadening (T₂, motion).

**Conditions.** Valid for a **symmetric** line with no resolved anisotropic
structure — a spin-½ resonance, or a quadrupolar/CSA pattern already collapsed by
fast MAS to a symmetric envelope.

**Limitations.** Empirical: the mixing has no microscopic meaning, and it **cannot
reproduce the asymmetric high-field tail** of a distributed quadrupolar site — for
that you need Czjzek. Fitting a glassy ²⁷Al line with a Gaussian systematically
misassigns the tail intensity to spurious 5- or 6-coordinate sites (see § Czjzek).

### Voigt (true) — `voigt`

**What it is.** The exact convolution of a Gaussian with a Lorentzian, with
**independent** widths:

$$V(x) = (G * L)(x) = \int G(x^\prime)\,L(x-x^\prime)\,dx^\prime$$

evaluated via `scipy.special.voigt_profile`. Unlike the pseudo-Voigt it keeps the
Gaussian and Lorentzian FWHM as two separate parameters, so a fit can genuinely
partition the broadening.

**History & literature.** W. Voigt (1912); the profile is standard in spectroscopy
wherever a homogeneous (Lorentzian) line is broadened by an inhomogeneous
(Gaussian) distribution.

**Use it for** cases where the two broadening mechanisms are physically distinct
and you want to report both — e.g. a T₂-limited Lorentzian width under a Gaussian
distribution of shifts.

**Limitations.** Still a symmetric line; no anisotropy. The Gaussian and Lorentzian
widths can be correlated in a noisy fit — constrain one if the data can't separate
them.

### J-multiplet — `jmultiplet`

**What it is.** Scalar (J) coupling to **n equivalent spin-½ neighbours** splits a
line into **n + 1** components at spacing J (Hz), with **binomial** intensities
(1 : n : n(n−1)/2 : …), each component a pseudo-Voigt of width `fwhm`. Component
$k$ ($k = 0\ldots n$) sits at $x_0 + (k - n/2)\,J$ with relative intensity
$\binom{n}{k}/2^n$.

**Literature.** Standard scalar-coupling multiplet theory; provided as dmfit's
*Jmultiplet*.

**Use it for** resolved J-coupling — e.g. ³¹P or ²⁹Si coupled to a fixed number of
equivalent spin-½ partners.

**Limitations.** Assumes strictly equivalent neighbours and weak (first-order)
coupling; unequal couplings or second-order effects are not modelled.

### Spinning sidebands — `sidebands`

**What it is.** A centreband plus a manifold of sidebands at **±k·ν_rot** (k = 1…
n_ssb), with a geometric intensity ratio `r` between successive orders; each band a
pseudo-Voigt. The sideband spacing uses the **actual** MAS rate ν_rot from the
experiment parameters, so the manifold is placed physically, not by hand (this
replaces dmfit's manually-positioned *ss band* lines).

**Literature.** Herzfeld & Berger, *J. Chem. Phys.* **73**, 6021 (1980), for the
sideband-intensity analysis of MAS.

**Use it for** a quick empirical description of a sideband pattern when you don't
need to invert it to a full CSA/quadrupolar tensor. For a *physical* sideband
manifold from a shielding tensor, use **CSA powder** instead, which generates the
sidebands from ζ and η.

**Limitations.** The geometric ratio is a convenience, not a tensor — it doesn't
encode ζ/η. Use it for bookkeeping, not for extracting anisotropy.

### Function fit — `function`

An arbitrary user expression **y = f(x; a, b, c, d)** (safe-evaluated), following
ssNake's *function fit*. For custom or didactic shapes that no physical model
covers. Purely empirical by definition.

### Spectrum (background) — `spectrum`

Not a lineshape but a **measured spectrum** used as a fit component, with a free
amplitude and a small shift. Use it to subtract a known impurity or a separately
measured phase whose lineshape you don't want to parameterise. The number that
comes out is the amount of that reference present, not a set of NMR parameters.

## 2 · Quadrupolar lineshapes (ordered sites)

For a half-integer quadrupolar nucleus of spin $I$, the quadrupolar coupling to
the electric-field-gradient (EFG) tensor $\mathbf{V}$ is

$$H_Q = \frac{eQ}{2I(2I-1)\hbar}\,\hat{I}\cdot\mathbf{V}\cdot\hat{I}, \qquad C_Q = \frac{eQ\,V_{zz}}{h}, \qquad \eta = \frac{V_{xx}-V_{yy}}{V_{zz}}$$

with $|V_{xx}| \leq |V_{yy}| \leq |V_{zz}|$ the sorted EFG eigenvalues
($0 \leq \eta \leq 1$).

### Quad CT, 2nd order — `quad_ct`

**What it is.** The **central transition** (+½ ↔ −½) is unaffected by the
quadrupolar interaction to first order; the observable powder pattern comes from
the **second-order** term. Its isotropic (MAS) shift and characteristic
"two-horned" powder shape are set by $C_Q$, $\eta$ and the Larmor frequency
$\nu_0$. The second-order shift of the central-transition centre of gravity is

$$\delta_{cg}^{(2)} = -\frac{3}{40}\,\frac{P_Q^2}{\nu_0^2}\,\frac{I(I+1)-\frac{3}{4}}{I^2(2I-1)^2}\times 10^6~\mathrm{ppm}, \qquad P_Q = C_Q\sqrt{1+\eta^2/3}$$

where $P_Q$ is the **quadrupolar product**; so the shift scales as
$(P_Q/\nu_0)^2$ — it shrinks at high field, which is why
quadrupolar sites are better resolved at high B₀. The full powder pattern (not just
its centre of gravity) is generated by mrsimulator.

**Literature.** Second-order quadrupolar theory: e.g. P. P. Man, "Quadrupolar
Interactions", in *eMagRes*; Freude & Haase, *Quadrupole Effects in Solid-State
NMR* (1993). Implemented in dmfit as *Q. 2nd order* (Massiot et al. 2002).

**Use it for** a **crystalline / well-defined** half-integer site (²⁷Al, ²³Na,
¹¹B, ⁸⁷Rb, …). Under MAS the second-order pattern narrows but does **not** vanish.

**Limitations.** Assumes a single, sharp (C_Q, η). If the site is disordered, this
model's sharp singularities won't match the smeared, tailing line — that is exactly
the regime Czjzek was built for.

### Quad 1st order (satellites) — `quad_first`

The **satellite transitions** (±3/2 ↔ ±1/2, …) split to **first order** in the
quadrupolar coupling, producing a wide manifold whose MAS spinning sidebands map
the EFG. LARMOR builds the full satellite manifold with its sideband structure
(dmfit's *quad 1st order*).

**Use it for** experiments where satellites are observed (small-to-moderate C_Q,
e.g. some ²³Na, ⁷Li, ²H-like cases). **Limitations:** needs the satellites to be
excited and a known spin rate; large C_Q pushes satellites out of range.

### Quad CT + CSA — `quad_csa`

A single site carrying **both** a second-order quadrupolar interaction *and* a
shielding anisotropy, with their relative orientation. Both tensors are simulated
together (mrsimulator).

**Use it for** nuclei where neither interaction is negligible (e.g. ⁵¹V, ⁷¹Ga,
some ⁸⁷Rb / ²⁷Al at high field). **Limitations:** more parameters (C_Q, η_Q, ζ,
η_CS and their relative orientation) — needs multi-field or high-quality data to
avoid correlations.

## 3 · Disordered quadrupolar lineshapes

### Czjzek — `czjzek`  ★ the glass model

**The problem it solves.** In a glass or amorphous solid, structural disorder at
the scale of two-to-three coordination shells makes the EFG — and therefore the
quadrupolar interaction — **different at every site**. The powder spectrum is then a
*distribution* of quadrupolar patterns superimposed on the usual powder average,
and it characteristically **tails toward high field (low ppm)**, because the
second-order shift is unidirectional and proportional to the interaction strength.
A single (C_Q, η) is meaningless here, and — crucially — a **Gaussian broadening of
a quadrupolar line does not reproduce this asymmetric tail** (d'Espinose de
Lacaillerie et al. 2008, Fig. 1).

**The model.** Czjzek's insight (from Mössbauer spectroscopy) is that the **five
independent components of the traceless, symmetric EFG tensor** — not C_Q and η —
are what add up from many independent structural contributions. By the **Central
Limit Theorem** those five components tend to *independent, identically distributed
Gaussians* of width σ, provided the solid is statistically isotropic. Transforming
that 5-D Gaussian to the NMR variables ($\nu_Q$, $\eta$) gives the **joint**
probability density (d'Espinose de Lacaillerie et al. 2008, Eq. 6; $d = 5$):

$$p(\nu_Q,\eta) = \frac{1}{\sqrt{2\pi}\,\sigma^5}\;\nu_Q^4\;\eta\left(1-\frac{\eta^2}{9}\right)\exp\!\left[-\frac{\nu_Q^2\,(1+\eta^2/3)}{2\sigma^2}\right]$$

Two features matter. First, ν_Q and η are **not independent** — they are coupled
through the tensor, exactly as physics demands (a distortion that changes the
strength also changes the asymmetry). Second, the whole distribution is governed by
a **single width $\sigma$**. That width fixes the mean-square quadrupolar product
(their Eq. 7):

$$\left\langle C_Q^2\,(1+\eta^2/3)\right\rangle = \left\langle P_Q^2\right\rangle \;\propto\; \sigma^2$$

so the one quantity you can honestly extract from a glass is the **root-mean-square
quadrupolar product √⟨P_Q²⟩** — a rotational invariant — together with the
isotropic chemical shift. (Their worked ²⁷Al slag glass: δ_iso = 72.7 ppm,
σ = 467 kHz ⟶ √⟨P_Q²⟩ = 6.96 MHz, from a *single* Czjzek line.)

The general Czjzek replaces the exponent 4 by **d − 1**, where **d = 5** is the
fully-isotropic case; a smaller d describes a more constrained (partially ordered)
distribution.

**Assumptions (and therefore conditions of validity)** — the four points of
d'Espinose de Lacaillerie et al. 2008:

1. structural elements contribute **additively** to the EFG;
2. those structural variables are **independent**;
3. the number of contributing neighbours is **sufficiently large**;
4. the ensemble is **statistically isotropic** (no order beyond the range of the
   quadrupolar interaction).

Points (2) and (4) restrict Czjzek to matter **not ordered beyond the quadrupolar
range** — i.e. genuinely disordered/amorphous solids. Point (3) is the most
demanding in practice: it needs a **coordination number of ~4 or more** around the
probe. This is why Czjzek works beautifully for four-coordinate glassy ²⁷Al but is
**not appropriate for ¹⁷O in oxide glasses** (two-coordinate O has too few
contributors for the CLT to hold).

**How it's used in LARMOR.** The Czjzek line has four parameters:

- **pos** — isotropic chemical shift δ_iso (ppm);
- **σ (sigma)** — the Czjzek width (LARMOR's `sigma_Cq_MHz`). Relation to dmfit:
  dmfit's *sCZ_CQ* ≈ 2σ; the mode of |C_Q| ≈ 2σ;
- **dCS** — an *independent* Gaussian width of the isotropic-**chemical-shift**
  distribution (dmfit's *FWHM CS*). Glasses disorder the shift as well as the EFG,
  and the two are separate physical effects — so LARMOR fits them separately. In a
  2D MQMAS map dCS elongates the peak **along the diagonal (CS axis)** while σ
  spreads it **along the quadrupolar axis**;
- **line** — a small round point/instrumental broadening (dmfit's *wid*).

The (C_Q, η) basis is simulated once by mrsimulator and re-weighted by the Czjzek
p(ν_Q, η) on every fit iteration, so the fit is fast.

**Literature.**
- G. Czjzek, J. Fink, F. Götz, H. Schmidt, J. M. D. Coey, J.-P. Rebouillat, A.
  Liénard, "Atomic coordination and the distribution of electric field gradients
  in amorphous solids", *Phys. Rev. B* **23**, 2513 (1981) — the original model.
- **J.-B. d'Espinose de Lacaillerie, C. Fretigny, D. Massiot, "MAS NMR spectra of
  quadrupolar nuclei in disordered solids: The Czjzek model", *J. Magn. Reson.*
  **192**, 244–251 (2008)**, doi:10.1016/j.jmr.2008.03.001 — the definitive NMR
  derivation, validity assumptions, and worked ²⁷Al examples used above.
- Application to aluminosilicate glasses: e.g. Neuville, Cormier, Massiot,
  *Geochim. Cosmochim. Acta* **68**, 5071 (2004).

**Limitations.** Reports only the *invariants* (δ_iso, √⟨P_Q²⟩ via σ) — individual
C_Q and η are, by construction, not defined for a distribution. Fails when its
assumptions fail: low coordination number (¹⁷O glass), or residual medium-range
order that breaks statistical isotropy. When there *is* a dominant, only mildly
perturbed tensor, use the extended Czjzek instead.

### ext. Czjzek — `ext_czjzek`

**What it is.** The **extended Czjzek** (Gaussian-perturbed) model: a **dominant**
EFG tensor (C_Q, η) plus a random Gaussian perturbation whose relative magnitude is
ε. It bridges the ordered limit (ε → 0, a sharp Quad-CT site) and the fully-
disordered Czjzek limit (large ε, or a vanishing dominant tensor). Physically it
describes a **partially ordered** environment — a defect site, a distorted
crystalline site, or a glass with a preferred local geometry.

**Literature.** G. Le Caër & R. A. Brand, *J. Phys.: Condens. Matter* **10**, 10715
(1998); **G. Le Caër, B. Bureau, D. Massiot, "An extension of the Czjzek model …
and its application to NMR spectroscopy", *J. Phys.: Condens. Matter* **22**,
065402 (2010)**.

**Use it for** sites intermediate between crystalline and glassy — e.g. defective
spinels, or a glass site with a well-defined dominant coordination. **Parameters:**
pos, dominant C_Q and η, the perturbation fraction **ε**, plus dCS and line as for
Czjzek. **Limitations:** more parameters than Czjzek; ε and the dominant tensor can
trade off unless the data are good. If ε is large the fit is just Czjzek — prefer
the simpler model then.

## 4 · Shielding-anisotropy (CSA) lineshapes

### CSA powder (MAS/static) — `csa_mas`

**What it is.** The spin-½ **chemical-shift-anisotropy** powder pattern from the
shielding tensor, in the **Haeberlen** convention: isotropic shift δ_iso, anisotropy
ζ (or δ_aniso) and asymmetry η_CS. Under MAS the static pattern breaks into a
centreband + spinning sidebands whose envelope traces the tensor; a spin rate of 0
gives the static powder pattern. Generated physically by mrsimulator (this replaces
hand-placed *ss band* lines).

**Literature.** Haeberlen convention: U. Haeberlen, *High Resolution NMR in Solids:
Selective Averaging* (1976). Sideband analysis: Herzfeld & Berger, *J. Chem. Phys.*
**73**, 6021 (1980).

**Use it for** spin-½ nuclei with resolved CSA — ³¹P, ¹³C, ²⁹Si, ¹¹⁹Sn, ²⁰⁷Pb.
**Limitations:** spin-½ only; needs the sideband manifold (moderate spin rate) to
constrain ζ and η_CS.

### CSA distribution (disordered) — `csa_czjzek`

**What it is.** The **CSA analogue of the Czjzek**: the shielding powder pattern
averaged over a **Gaussian distribution of the anisotropy ζ** (width σ_ζ), for a
glassy/amorphous spin-½ site whose shielding is itself disordered. When σ_ζ → 0 it
collapses to the ordinary CSA powder pattern.

**Use it for** amorphous spin-½ systems (e.g. ³¹P or ²⁹Si in glasses) where the
anisotropy is distributed, not single-valued. **Parameters:** pos, mean ζ, the
distribution width **σ_ζ**, η, and a Gaussian broadening. **Limitations:** a
pragmatic Gaussian-on-ζ model (not derived from a 5-D CLT argument like the
quadrupolar Czjzek); report the mean and width of ζ, not a single value.

## Conventions

**Quadrupolar coupling.** C_Q = eQV_zz/h (MHz); η = (V_xx−V_yy)/V_zz ∈ [0,1]. The
rotational invariant is the **quadrupolar product** P_Q = C_Q·√(1+η²/3); for a
disordered site only P_Q (and its RMS over the distribution) is meaningful.
ν_Q here denotes the EFG-strength variable of the Czjzek PDF.

**Czjzek width.** LARMOR's `sigma_Cq_MHz` = σ, the width of the EFG-component
Gaussian expressed in C_Q units. dmfit relation: **sCZ_CQ ≈ 2σ**; mode of |C_Q| ≈
2σ. Dimensionality d = 5 for a fully isotropic (standard Czjzek) distribution;
mrsimulator's `CzjzekDistribution` implements this d = 5 case, which is also
dmfit's default (d = 5).

**Shielding (Haeberlen).** δ_iso = (δ_xx+δ_yy+δ_zz)/3; anisotropy ζ = δ_zz − δ_iso;
asymmetry η_CS = (δ_yy − δ_xx)/ζ, with |δ_zz−δ_iso| ≥ |δ_xx−δ_iso| ≥ |δ_yy−δ_iso|.

**Widths.** LARMOR accepts a line width in **ppm or Hz** in the parameter table
(e.g. `300Hz`, `1.5kHz`, `2ppm`); it converts to the native unit using the Larmor
frequency.

**MQMAS.** The 2D isotropic (F1) axis and the CS/quadrupolar reference axes are
documented in the MQMAS manual (**? ▸ User manuals ▸ MQMAS**).
